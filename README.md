### Sobre

Um formulario simples com validação nos campos obrigaórios e notificação de erros e retorno de sucesso.

### Desenvolvimento

Frontend:
- TypeScript.
- React.

### Instalação 

Para rodar esse projeto, existe duas formas: 

1 - Rodando localmente:  
``` 
cd frontend
npm install // para instalar as dependências
npm start // para inicializar a pagina.
``` 

### Requisitos da Aplicação 

Cadastro do formulário seguindo o seguinte modelo (* campos obrigatórios): 
  - Contato 
    - Nome*
    - Data Inicial*
    - Data Final*
    - Proprietário*
    - Laboratório*
    - Observações

### Requisitos dos Testes
Rode o `npm test` para ver os testes.
